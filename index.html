Angular promises demo

In Angular the $q is a service that helps you run functions asynchronously, and use their return values (or exceptions) when they are done processing.

Demo with timer

Here we are going to simulate an asynchronous operation with a timer. The timer will run for the duration which the asynchronous operation (like a HTTP call, file read/write, database read/write etc.) would take.

The operation duration will be 500ms, after which a random string will be outputted.
